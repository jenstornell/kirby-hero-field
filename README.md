# Kirby Hero Field

*Version 2.0*

Large image preview below the field. Best suited for wide images.

**Features**

- Multi language support.
- Multi instances support.

## Setup

### 1. Blueprint

To make it work as expected, add the following code to your blueprint:

```
fields:
  yourfield:
    label: 
    type: hero
```
